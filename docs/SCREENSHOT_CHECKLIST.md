# Screenshot Checklist for v1.0.0-beta

Please capture the following screenshots on a physical device (or emulator) running the latest v1.0.0-beta build. Place them in `docs/images/`.

## Required Screenshots

1.  **Discovery Screen**
    *   **Filename:** `discovery_screen.png`
    *   **State:** Show the list of available devices (unpaired).
    *   **Caption:** "Main screen showing discovered devices."

2.  **Unpaired Device Detail**
    *   **Filename:** `unpaired_device.png`
    *   **State:** Tap on an unpaired device. Show the "Request Pairing" button.
    *   **Caption:** "Pairing request screen."

3.  **Paired Device Detail (Connected)**
    *   **Filename:** `paired_device.png`
    *   **State:** A fully paired and connected device. Show the list of plugins (Battery, Multimedia, etc.).
    *   **Caption:** "Device details showing status and plugins."

4.  **Plugin Settings (Example)**
    *   **Filename:** `plugin_settings.png`
    *   **State:** Inside the settings for a plugin (e.g., Notification Filter or Share).
    *   **Caption:** "Configuring plugin options."

5.  **Run Command Plugin**
    *   **Filename:** `run_command.png`
    *   **State:** The list of available remote commands.
    *   **Caption:** "Executing remote commands."

## Style Guidelines
*   **Theme:** Use **Light Mode** for consistency (or provide both Light/Dark sets).
*   **Device Frame:** Optional, but clean screenshots (no status bar clutter) are preferred.
*   **Language:** English.
